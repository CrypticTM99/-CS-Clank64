Clank has arrived from another galaxy!

This time, they are on a secret mission to take on Bowser in SM64!

Uses a custom moveset + ECM config

To enable the ECM config, make sure to have Easy-custom-moveset enabled!

Comes with sounds from the games Ratchet & Clank going commando & Deadlocked

Note: Clank can attack friendly toads & pink bombs!

Custom moves:
- Flutter step
- Special animation from ECM for jumps
- Wall ride jump
- Backflip to helicopter ! Clank can glide for a long time!
- Rocket jump (at least twice per level)
- and more!

How to install:

    Extract the zip
    Drag & drop the mod folder [CS] Clank64 v1 inside your [mods] folder
    Remember to enable the mod by selecting it within the mod menu and make sure character select is ON
    Enjoy!
